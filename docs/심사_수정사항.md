# App Store / Play Store 심사 수정사항

## 1. API 키 하드코딩 제거 (보안)

소스코드에 직접 작성되어 있던 모든 API 키를 환경변수 방식으로 변경했습니다.

**변경 방식**: `--dart-define-from-file=.env.json`

| 서비스 | 수정 파일 |
|--------|-----------|
| Supabase URL/Key | `lib/config/supabase_config.dart` |
| Supabase (백그라운드 중복) | `lib/services/background_location_service.dart` |
| Kakao Native/JS Key | `lib/config/kakao_config.dart` |
| Kakao (auth 중복) | `lib/providers/auth_provider.dart` |
| OpenWeatherMap Key | `lib/services/weather_service.dart` |
| Naver Client ID/Secret | `lib/services/blog_service.dart` |
| Supabase Storage URL | `lib/models/oreum_model.dart`, `oreum_image_model.dart`, `stamp_provider.dart` |

**추가된 파일**:
- `lib/config/env_config.dart` — 환경변수 참조 클래스
- `.env.json` — 실제 키 값 (`.gitignore`에 추가됨)
- `.env.example.json` — 빈 템플릿 (커밋용)

**빌드 명령어**:
```
flutter run --dart-define-from-file=.env.json
flutter build ios --dart-define-from-file=.env.json
flutter build apk --dart-define-from-file=.env.json
```

## 2. Android 보안 설정 수정

`android/app/src/main/AndroidManifest.xml`

```xml
<!-- 변경 전 -->
android:usesCleartextTraffic="true"

<!-- 변경 후 -->
android:usesCleartextTraffic="false"
```

HTTP 평문 통신을 차단하여 Play Store 보안 정책을 충족합니다.

## 3. iOS 권한 문구 수정

`ios/Runner/Info.plist`

**추가**:
```xml
<key>NSPhotoLibraryAddUsageDescription</key>
<string>등산 사진을 저장하기 위해 사진 라이브러리 저장 권한이 필요합니다.</string>
```

**삭제** (deprecated):
```xml
<key>NSLocationAlwaysUsageDescription</key>
<string>등산 중 백그라운드에서도 경로를 기록하기 위해 위치 권한이 필요합니다.</string>
```

`NSLocationAlwaysUsageDescription`은 iOS 11 이후 deprecated되었으며, `NSLocationAlwaysAndWhenInUseUsageDescription`이 이를 대체합니다 (기존 유지).
