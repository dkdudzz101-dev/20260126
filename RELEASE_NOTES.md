버전 1.3.0 업데이트

제주도 368개 오름을 탐험하고 기록하는 오름 등반 앱!

[앱 소개]
제주 오름 탐험 앱은 제주도의 모든 오름 정보를 한눈에 확인하고,
GPS 기반 등산 기록과 스탬프 수집으로 나만의 오름 도감을 완성하는 앱입니다.

[주요 기능]

▶ 오름 정보
• 184개 오름의 난이도, 거리, 소요시간, 해발고도 상세 정보
• 테마별 탐색: 일출명소, 일몰명소, 대표명소, 전망좋은, 숲속힐링, 가족추천
• 오름별 사진 갤러리 (공식 사진 + 커뮤니티 사진)
• 등산로 정보, 시설물 안내, 계절별 추천, 접근 제한 정보
• 관련 블로그 후기 연동

▶ 카카오맵 연동 지도
• 368개 오름 위치를 지도에서 한눈에 확인
• 줌 레벨에 따른 마커 클러스터링
• 등산로 경로 시각화 및 시설물 마커 표시
• 정상 인증 구역(100m) 표시
• 내 위치 기반 가까운 오름 찾기
• 실시간 제주 날씨 및 등산 적합도 표시
• 북마크한 오름만 필터링

▶ GPS 등산 기록
• 실시간 GPS 추적으로 등산 경로 기록
• 걸음수, 이동거리, 남은거리, 예상 남은시간 실시간 표시
• 고도 상승/하강량 측정 (노이즈 필터링 적용)
• 등산 중 사진 촬영 및 기록 첨부
• 정상 50m 접근 시 자동 알림
• 일시정지/재개 기능
• 하산 모드 지원 (등산/하산 구간 별도 기록)

▶ 스탬프 수집
• 정상 100m 이내 GPS 인증으로 스탬프 획득
• 스탬프북 그리드로 수집 현황 한눈에 확인
• 전체 오름 대비 달성률 표시
• 근처 미수집 오름 바로가기

▶ 뱃지 시스템
• 1, 5, 10, 30, 100, 368개 달성 등 다양한 뱃지
• 거리, 시간, 연속 등반, 커뮤니티 참여 등 카테고리별 뱃지
• 달성 진행률 표시

▶ 등산 공유
• 사진 + 경로 + 통계 카드 (삼성헬스 스타일)
• 경로 + 통계 카드
• 기본 통계 카드
• SNS 공유 기능 (#제주오름 #등산 #오름탐험)

▶ 커뮤니티
• 등반완료, 후기, 질문, 동행모집 카테고리별 게시글
• 인기순, 최신순, 내 글 필터링
• 오름별 게시글 필터링
• 사진 첨부 (최대 5장), 좋아요, 댓글
• 사용자 레벨 표시

▶ 프로필 & 통계
• 완등 오름 수, 총 이동거리, 총 걸음수 대시보드
• 등산 기록 히스토리
• 뱃지 컬렉션
• 프로필 사진 및 닉네임 설정

▶ 소셜 로그인
• 카카오, 네이버, Apple 로그인 지원
• 클라우드 동기화로 데이터 안전 보관

▶ 날씨 & 추천
• 실시간 제주 날씨 표시 (기온, 습도, 풍속, 체감온도)
• 날씨 기반 오름 추천 (비/흐림/더위/맑음별 맞춤 코스)
• 일출/일몰 시간 표시

[v1.3.0 업데이트 내용]

▶ 새 기능
• 하산 모드 추가 - 정상 도착 후 하산 구간을 별도로 기록 (거리/시간/걸음수/칼로리)
• 입구 도착 자동 감지 - 하산 중 출발점 100m 이내 접근 시 하산 완료 알림
• 오름 정보 오류 신고 기능 추가 (입구/정상 좌표, 주차장, 화장실, 등산로 오류 신고 가능)
• 오름별 방문팁 및 등산로 상태 정보 표시

▶ 버그 수정
• 정상 도착 알림이 여러 번 표시되던 문제 수정
• 등산 시작 시 백그라운드 위치 권한 강제 요청 제거
• 등반 완료 버튼 중복 실행 방지
